# Scripts

This directory contains utility scripts for development, testing, and deployment.

## Structure

- **docker/** - Docker-related scripts
  - `test-docker.sh` - Test Docker setup
  - `validate-docker.sh` - Validate Docker configuration

- **dev/** - Development scripts (reserved for future use)

## Usage

### Docker Scripts

```bash
# Test Docker setup
./scripts/docker/test-docker.sh

# Validate Docker configuration
./scripts/docker/validate-docker.sh
```
